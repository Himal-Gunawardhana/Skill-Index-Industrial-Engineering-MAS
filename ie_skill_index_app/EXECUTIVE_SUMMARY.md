- **Mobile Application:** Android 5.0+ (47 MB APK)

## Quick Start Guide

### For IE Users

1. Login with email/password
2. Create new assessment
3. Select style, operation, shift, module
4. Use timers (sequential unlock)
5. Enter good garments count
6. Get instant results

### For Admins

1. Long-press "IE Skill Index" title
2. Enter admin credentials
3. Select admin user & authenticate
4. Manage styles and operations (with SMV values)
